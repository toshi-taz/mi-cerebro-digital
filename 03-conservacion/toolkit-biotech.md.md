# Toolkit Bio-Tech para Conservación

## Herramientas activas
- iNaturalist: registro de avistamientos en campo
- pyinaturalist: acceso a API de iNaturalist desde Python
- QGIS: mapeo y análisis geoespacial (gratuito)
- pandas / matplotlib: análisis y visualización de datos

## Recursos de aprendizaje
- pyinaturalist docs: pyinaturalist.readthedocs.io
- QGIS tutorials: docs.qgis.org
- iNaturalist API: api.inaturalist.org
